# Dotfiles (macOS)

These are my personal dotfiles that I use to improve my daily routine as a software developer on macOS.

## Tools

Below are some tools that I use.

* [zsh and oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh/wiki)
* [zsh plugins](https://github.com/ohmyzsh/ohmyzsh/wiki/Plugins)
* [Neovim](https://neovim.io/)
* [lazy.nvim](https://github.com/folke/lazy.nvim)

## Packages

My favorite packages and tools on macOS.

* [Homebrew](https://brew.sh/)
* [nvm](https://github.com/nvm-sh/nvm)
* [pyenv](https://github.com/pyenv/pyenv)
* [rbenv](https://github.com/rbenv/rbenv)
* [SDKMAN](https://sdkman.io/)
* [fzf](https://github.com/junegunn/fzf)

## Neovim Plugins

* [catppuccin](https://github.com/catppuccin/nvim) — color scheme
* [Telescope](https://github.com/nvim-telescope/telescope.nvim) — fuzzy finder
* [Treesitter](https://github.com/nvim-treesitter/nvim-treesitter) — syntax highlighting
* [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) + [Mason](https://github.com/williamboman/mason.nvim) — LSP
* [nvim-cmp](https://github.com/hrsh7th/nvim-cmp) — autocomplete
* [Lazygit](https://github.com/kdheepak/lazygit.nvim) — git UI inside Neovim
* [lualine](https://github.com/nvim-lualine/lualine.nvim) — status bar
